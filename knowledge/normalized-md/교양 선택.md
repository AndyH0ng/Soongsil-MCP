### 2023학년도 이후 입학자 교양선택(BALANCE)

| 영역 | 분류 |
| --- | --- |
| 1영역 | 인간·언어 |
| 2영역 | 문화·예술 |
| 3영역 | 사회·정치·경제 |
| 4영역 | 과학·기술 (Bridge 교과: 수리·물리·화학·생물) |
| 5영역 | 자기개발·진로탐색 |

- 이수요건: 1~5영역 중 3개 영역 이상 이수

### 2021~2022학년도 입학자 교양선택 이수체계

| 교과구분 | 영역 | 최소 이수요건 |
| --- | --- | --- |
| 숭실품성교과 | 인성과 리더십 / 자기계발과 진로탐색 | 1개 영역 선택, 1과목 필수 |
| 기초역량교과 | 한국어의사소통 / 국제어문 | 1개 영역 선택, 1과목 필수 |
| 균형교양교과 | 문학·예술 / 역사·철학·종교 / 정치·경제·경영 / 사회·문화·심리 / 자연과학·공학·기술 | 2개 영역 선택, 각 1과목(총 2과목) 필수 |

- 2020학년도 이전 입학자는 입학년도 기준 이수체계 또는 2021~2022학년도 이수체계 중 선택 가능
