# QA Checklists

## 학칙 일반 질의응답

1. 질문 핵심 키워드를 추출한다. (예: 휴학, 복학, 자퇴, 전과, 수강신청)
2. `references/law-topic-index.md`에서 1차/2차 조문을 라우팅한다.
3. `references/law-articles.md`에서 해당 조문 카드와 인덱스를 확인한다.
4. 숫자 판단이 포함되면 `references/law-numeric-rules.md`를 추가 확인한다.
5. `knowledge/normalized-md/학칙.md`에서 후보를 찾고, 필요시 `knowledge/raw-md/학칙.raw.md`를 대조한다.
6. `학칙.pdf`로 최종 재검증한다.
7. 답변을 조문 단위로 요약하고 문서/페이지를 모두 인용한다.
8. 해석이 갈릴 수 있으면 가능한 해석을 병기한다.

## 졸업요건 충족 판정

1. 입학년도/학과/전공형태를 먼저 확정한다.
2. `knowledge/normalized-md/학점 이수 체계.md`에서 기준 학점을 찾는다.
3. `knowledge/normalized-md/교양 필수.md`, `knowledge/normalized-md/교양 선택.md`와 대조한다.
4. 최종 숫자를 PDF 페이지와 대조한다.
5. 영역별 부족 학점을 계산한다.
6. "지금 졸업 가능/불가"를 명시한다.

## 재수강 가능/영향 분석

1. `knowledge/normalized-md`에서 `재수강`, `성적`, `학점` 키워드로 먼저 검색한다.
2. 재수강 운영 원칙이 있는지 확인한다.
3. 필요시 PDF를 추가 대조한다.
4. PDF에서 페이지를 재검증한다.
5. 횟수/상한 정보가 없으면 없다고 명시한다.
6. 확인된 사실과 미확인 사항을 구분해 제시한다.

## 장학 역치 비교

1. 장학 기준 문서가 있는지 먼저 확인한다.
2. 기준이 없으면 비교를 중단하고 문서 요청을 한다.
3. 기준이 있으면 normalized-md에서 후보 조항을 찾고 PDF 페이지를 재검증한다.
4. 평점/이수학점을 기준과 비교한다.
5. 가능성 판단과 남은 조건을 수치로 제시한다.

## 수강신청 보조

1. `knowledge/normalized-md/학사 일정.md`에서 신청/변경/취소 기간을 찾는다.
2. PDF 페이지로 기간을 재확인한다.
3. 부족 영역 학점을 기준으로 우선 과목군을 정한다.
4. 마감일 기준 실행 순서를 제시한다.
